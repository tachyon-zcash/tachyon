# [ZIP-317](https://zips.z.cash/zip-0317)

**Tracking:** [#111](https://github.com/tachyon-zcash/tachyon/issues/111)

**('Update' ZIP, Category 'Consensus')**

## I. Dependencies

- [Tachyon Shielded Protocol (#103)](https://github.com/tachyon-zcash/tachyon/issues/103) defines tachyactions and `nActionsTachyon`

## II. Design Considerations

ZIP-317 prices a transaction by its `logical_actions` — a protocol-neutral count of the verification and state cost it imposes on validators, summed across the transparent, Sprout, Sapling, Orchard, ZSA, and memo contributions (see [Fee Calculation](https://zips.z.cash/zip-0317#fee-calculation)). Tachyon adds a bundle type ZIP-317 does not account for: tachyactions, carried under an aggregation model where the same logical work is split across an adjunct (which carries the actions) and the aggregate (which carries the covering proof). The open question is how to define `contribution_Tachyon` — this is still under discussion and nothing has been decided yet.
