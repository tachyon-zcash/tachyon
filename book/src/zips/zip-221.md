# [ZIP-221](https://zips.z.cash/zip-0221)

**Tracking:** [#108](https://github.com/tachyon-zcash/tachyon/issues/108)

**('Update' ZIP, Category 'Consensus')**

This touches on where the per-block Tachyon pool state gets committed so that light clients can verify it. Block headers have a field called **`hashChainHistoryRoot`** (commitment to a merkle mountain range, MMR), and we'll need to extend its leaf definition to support Tachyon, ideally.
