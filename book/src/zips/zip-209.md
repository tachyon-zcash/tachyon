# [ZIP-209](https://zips.z.cash/zip-0209)

**Tracking:** [#109](https://github.com/tachyon-zcash/tachyon/issues/109)

**('Update' ZIP, Category 'Consensus')**

## I. Dependencies

- [Tachyon Shielded Protocol (#103)](https://github.com/tachyon-zcash/tachyon/issues/103) defines `valueBalanceTachyon`

## II. Design Considerations

[ZIP-209](https://zips.z.cash/zip-0209) is a consensus rule — the turnstile — that tracks the running balance of each shielded pool across the block chain and rejects any block that would drive a pool's balance negative; it currently covers the Sprout, Sapling, and Orchard pools. Tachyon introduces a new shielded pool with a signed `valueBalanceTachyon` field, so the open question is how to extend ZIP-209 to cover it — this is still under discussion and nothing has been decided yet.
