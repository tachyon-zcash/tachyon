# Tachyon Bundle / Aggregate Transaction Format

**Tracking:** [#104](https://github.com/tachyon-zcash/tachyon/issues/104)

**('Additive / Update' ZIP, Category 'Consensus')**

This depends very much on the 'Extensible Transaction Format' ([ZIP-248](https://zips.z.cash/zip-0248)), the precise interpretation of that ZIP from the authors, and whether it lands. [librustzcash](https://github.com/zcash/librustzcash) would need to be rewritten to emit tachyon bundles as a ZIP-248-style bundle instead of a hardcoded v6 field, assuming the proposal lands.
