# Tachyon Aggregator Protocol

**Tracking:** [#106](https://github.com/tachyon-zcash/tachyon/issues/106)

**('Additive' ZIP, Category 'Network')**

Defines aggregator nodes (which sit between mempool and miners) and how they handle Autonomes / Aggregates / Adjuncts stamps. Also defines miners' roles and the block construction protocol.
